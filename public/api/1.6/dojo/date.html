dojo.date (version 1.6)

Object » dojo.date

Date manipulation utilities

Method Summary

  • dojo.date.add(date, interval, amount) returns Date Add to a Date in intervals of different size, from milliseconds to years
  • dojo.date.compare(date1, date2, portion) returns int Compare two date objects by date, time, or both.
  • dojo.date.difference(date1, date2, interval) returns Number (integer) Get the difference in a specific unit of time (e.g., number of months, weeks, days, etc.) between two dates, rounded to the nearest integer.
  • dojo.date.getDaysInMonth(dateObject) returns Number Returns the number of days in the month used by dateObject
  • dojo.date.getTimezoneName(dateObject) returns String Get the user's time zone as provided by the browser
  • dojo.date.isLeapYear(dateObject) returns Boolean Determines if the year of the dateObject is a leap year

dojo.date.isLeapYear
Defined by dojo.date
Returns Boolean

Leap years are years with an additional day YYYY-02-29, where the year number is a multiple of four with the following exception: If a year is a multiple of 100, then it is only a leap year if it is also a multiple of 400. For example, 1900 was not a leap year, but 2000 is one.
