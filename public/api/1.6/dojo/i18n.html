dojo.i18n (version 1.6)

Object » dojo.i18n
Defined in dojo/i18n.js

Utility classes to enable loading of resources for internationalization (i18n)

Method Summary

  • dojo.i18n._preloadLocalizations(bundlePrefix, localesGenerated) Load built, flattened resource bundles, if available for all locales used in the page. Only called by built layer files.
  • dojo.i18n._requireLocalization(moduleName, bundleName, locale, availableFlatLocales, m, b) See dojo.requireLocalization()
  • dojo.i18n._searchLocalePath(locale, down, searchFunc) A helper method to assist in searching for locale-based resources. Will iterate through the variants of a particular locale, either up or down, executing a callback function. For example, "en-us" and true will try "en-us" followed by "en" and finally "ROOT".
  • dojo.i18n.getLocalization()
  • dojo.i18n.normalizeLocale(locale) returns String Returns canonical form of locale, as used by Dojo.

Defined by dojo.i18n

All variants are case-insensitive and are separated by '-' as specified in RFC 3066. If no locale is specified, the dojo.locale is returned. dojo.locale is defined by the user agent's locale unless overridden by djConfig.
