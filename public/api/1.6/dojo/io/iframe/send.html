dojo.io.iframe.send (version 1.6)

dojo.require("dojo.io.iframe");
Defined in dojo/io/iframe.js

Usage:

var foo = new dojo.io.iframe.send(/* dojo.io.iframe.__ioArgs */ args);

Function that sends the request to the server. This transport can only process one send() request at a time, so if send() is called multiple times, it will queue up the calls and only process one at a time.
