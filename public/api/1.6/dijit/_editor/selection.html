Object » dijit._editor.selection
dojo.require("dijit._editor.selection");

Method Summary

  • dijit._editor.selection.collapse(beginning) Function to collapse (clear), the current selection
  • dijit._editor.selection.getAncestorElement(tagName) returns DOMNode Return the parent element of the current selection which is of type tagName (or one of the other specified tagName)
  • dijit._editor.selection.getParentElement() Get the parent element of the current selection
  • dijit._editor.selection.getParentOfType(node, tags) returns DOMNode Function to locate a parent node that matches one of a set of tags
  • dijit._editor.selection.getSelectedElement() Retrieves the selected element (if any), just in the case that a single element (object like and image or a table) is selected.
  • dijit._editor.selection.getSelectedHtml() returns String Return the html text of the current selection or null if unavailable
  • dijit._editor.selection.getSelectedText() returns String Return the text (no html tags) included in the current selection or null if no text is selected
  • dijit._editor.selection.getType() returns String Get the selection type (like dojo.doc.select.type in IE).
  • dijit._editor.selection.hasAncestorElement(tagName) returns Boolean Check whether current selection has a parent element which is of type tagName (or one of the other specified tagName)
  • dijit._editor.selection.inSelection(node) returns boolean This function determines if 'node' is in the current selection.
  • dijit._editor.selection.isTag(node, tags) returns String Function to determine if a node is one of an array of tags.
  • dijit._editor.selection.remove() returns Selection Function to delete the currently selected content from the document.
  • dijit._editor.selection.selectElement(element, nochangefocus) clear previous selection and select element (including all its children)
  • dijit._editor.selection.selectElementChildren(element, nochangefocus) clear previous selection and select the content of the node (excluding the node itself)
