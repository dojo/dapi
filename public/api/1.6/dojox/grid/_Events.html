dojox.grid._Events (version 1.6)

Object » dojox.grid._Events
dojo.require("dojox.grid._Events");
Defined in dojox/grid/_Events.js

Usage:

var foo = new dojox.grid._Events();

Default synthetic events dispatched for _Grid. dojo.connect to events to retain default implementation or override them for custom handling.

onStyleRow
Defined by dojox.grid._Events
Perform row styling on a given row. Called whenever row styling is updated.
ParameterTypeDescription
inRowObjectObject containing row state information: selected, true if the row is selcted; over: true of the mouse is over the row; odd: true if the row is odd. Use customClasses and customStyles to control row css classes and styles; both properties are strings.
Example 1
onStyleRow({ selected: true, over:true, odd:false })
