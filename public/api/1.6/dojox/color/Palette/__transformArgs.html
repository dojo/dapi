The keywords argument to be passed to the dojox.color.Palette.transform function. Note that

while all arguments are optional, some arguments must be passed. The basic concept is that you pass a delta value for a specific aspect of a color model (or multiple aspects of the same color model); for instance, if you wish to transform a palette based on the HSV color model, you would pass one of "dh", "ds", or "dv" as a value.

Property Summary

  • da The delta to be applied to the alpha aspect of the RGBA color model.
  • db The delta to be applied to the blue aspect of the RGB/RGBA color model.
  • dc The delta to be applied to the cyan aspect of the CMY/CMYK color model.
  • dg The delta to be applied to the green aspect of the RGB/RGBA color model.
  • dh The delta to be applied to the hue aspect of the HSL/HSV color model.
  • dk The delta to be applied to the black aspect of the CMYK color model.
  • dl The delta to be applied to the luminosity aspect of the HSL color model.
  • dm The delta to be applied to the magenta aspect of the CMY/CMYK color model.
  • dr The delta to be applied to the red aspect of the RGB/RGBA color model.
  • ds The delta to be applied to the saturation aspect of the HSL/HSV color model.
  • dv The delta to be applied to the value aspect of the HSV color model.
  • dy The delta to be applied to the yellow aspect of the CMY/CMYK color model.
  • use Specify the color model to use for the transformation. Can be "rgb", "rgba", "hsv", "hsl", "cmy", "cmyk".
