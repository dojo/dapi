dojox.fx._Complex (version 1.6)

Object » dojox.fx._Complex
dojo.require("dojox.fx.ext-dojo.complex");
Defined in dojox/fx/ext-dojo/complex.js

Usage:

var foo = new dojox.fx._Complex(/* */ options);

A class that takes a complex property such as clip style: rect(10px 30px 10px 50px), and breaks it into seperate animatable units. The object has a getValue() that will return a string with the modified units.

Method Summary

  • constructor(options) returns dojox.fx._Complex A reference to the constructor function for this object.
  • getNumAndUnits(prop) returns Object Helper function that returns the numeric verion of the string property (or dojo.Color object) and the unit in which it was defined.
  • getProps(str) returns String Helper function that splits a stringified set of properties into individual units.
  • getValue(r) returns String Returns a string with teh same integrity as the original star and end, but with the modified units.
