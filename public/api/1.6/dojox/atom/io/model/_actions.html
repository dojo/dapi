Object » dojox.atom.io.model._actions
dojo.require("dojox.atom.io.model");
Defined in dojox/atom/io/model.js

Container for tag handling functions. Each child of this container is a handler function for the given type of node. Each accepts two parameters: obj: Object. The object to insert data into. node: DOM Node. The dom node containing the data Google news Google news
