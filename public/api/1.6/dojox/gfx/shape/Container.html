dojox.gfx.shape.Container (version 1.6)

Object » dojox.gfx.shape.Container

a container of shapes, which can be used as a foundation for renderer-specific groups, or as a way to logically group shapes (e.g, to propagate matricies)

Method Summary

  • dojox.gfx.shape.Container._moveChildToBack(shape) returns self moves a shape to back of the list of shapes
  • dojox.gfx.shape.Container._moveChildToFront(shape) returns self moves a shape to front of the list of shapes
  • dojox.gfx.shape.Container.add(shape) returns self adds a shape to the list
  • dojox.gfx.shape.Container.closeBatch() submits the current batch, append all pending child shapes to DOM
  • dojox.gfx.shape.Container.openBatch() starts a new batch, subsequent new child shapes will be held in the batch instead of appending to the container directly
  • dojox.gfx.shape.Container.remove(shape, silently) returns self removes a shape from the list
