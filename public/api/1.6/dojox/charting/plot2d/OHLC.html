A plot that represents typical open/high/low/close (financial reporting, primarily). Unlike most charts, the Candlestick expects data points to be represented by an object of the form { x?, open, close, high, low, mid? }, where both x and mid are optional parameters. If x is not provided, the index of the data array is used.
