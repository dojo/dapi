A fast buffer for creating large strings.

Property Summary

  • length The current length of the internal string.

Method Summary

  • append(s) returns dojox.string.Builder Append all arguments to the end of the buffer
  • appendArray(strings) returns dojox.string.Builder Append an array of items to the internal buffer.
  • clear() returns dojox.string.Builder Remove all characters from the buffer.
  • concat(s) returns dojox.string.Builder Alias for append.
  • constructor(str) returns dojox.string.Builder A reference to the constructor function for this object.
  • insert(index, str) returns dojox.string.Builder Insert string str starting at index.
  • remove(start, len) returns dojox.string.Builder Remove len characters starting at index start. If len is not provided, the end of the string is assumed.
  • replace(oldStr, newStr) returns dojox.string.Builder Replace instances of one string with another in the buffer.
  • toString() returns String Return the string representation of the internal buffer.
