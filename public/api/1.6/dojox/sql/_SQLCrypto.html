dojox.sql._SQLCrypto (version 1.6)

dojo.require("dojox.sql._base");
Defined in dojox/sql/_base.js

Usage:

var foo = new dojox.sql._SQLCrypto(/* */ action, /* */ sql, /* */ password, /* */ args, /* */ callback);

A private class encapsulating any cryptography that must be done on a SQL statement. We instantiate this class and have it hold it's state so that we can potentially have several encryption operations happening at the same time by different SQL statements.
