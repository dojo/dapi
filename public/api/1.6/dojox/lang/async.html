dojox.lang.async (version 1.6)

Object » dojox.lang.async

Method Summary

  • dojox.lang.async.any(x) Executes functions in parallel. As soon as one of them finishes cancels the rest.
  • dojox.lang.async.ifThen(cond, ifTrue, ifFalse) Executes a condition, waits for it if necessary, and executes one of two functions.
  • dojox.lang.async.loop(cond, body) Executes a condition, waits for it if necessary, and executes the body, if truthy value was returned. Then it repeats the cycle until the condition function returns a falsy value.
  • dojox.lang.async.par(x) Executes functions in parallel. Waits for all of them to finish.
  • dojox.lang.async.select(cond, x) Executes a condition, waits for it if necessary, and executes Nth function from list.
  • dojox.lang.async.seq(x) Executes functions sequentially. Waits if any of them returns Deferred.
