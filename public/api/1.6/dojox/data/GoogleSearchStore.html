Usage:

var foo = new dojox.data.GoogleSearchStore(/* Object */ args);

The GoogleSearchStore is a Datastore interface to the Google search service. The constructor accepts the following arguments:

  • label - the label attribute to use. Defaults to titleNoFormatting
  • key - The API key to use. This is optional
  • lang - The language locale to use. Defaults to the browser locale
