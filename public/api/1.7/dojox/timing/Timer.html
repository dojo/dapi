Usage:

var foo = new dojox.timing.Timer(/* int */ interval);

Timer object executes an "onTick()" method repeatedly at a specified interval. repeatedly at a given interval.

Method Summary

  • constructor(interval) returns dojox.timing.Timer Interval between function calls, in milliseconds.
  • setInterval(interval) Reset the interval of a timer, whether running or not.
  • start() Start the timer ticking.
  • stop() Stop the timer.

Event Summary

  • onTick() Method called every time the interval passes. Override to do something useful.
