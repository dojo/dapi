dojox.lang.oo.rearrange
Defined by dojox.lang.oo
Returns Object: Object: the original object

Properties of an object are to be renamed or removed specified by "map" argument. Only own properties of "map" are processed.

ParameterTypeDescription
bagObject:the object to be processed
mapObject:the dictionary for renaming (false value indicates removal of the named property)
Example 1
	oo.rearrange(bag, {
		abc: "def",	// rename "abc" attribute to "def"
		ghi: null	// remove/hide "ghi" attribute
	});
